## How to use it

### Clone the reposityory

```
$ git clone git@github.com:pierrechls/web-midi-experiment.git
```

### Install dependencies

```
$ npm install
```

### Serve with hot reload

```
$ npm run dev
```

### Build for production

```
$ npm run build
```
